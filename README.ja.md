# address-japan

日本の都道府県、市区町村、町字のデータをCSVファイルとしてダウンロードできるプロジェクトです。

## 必要環境
このプロジェクトを利用するにはDeno環境が必要です。

## 使い方
```
$ mkdir data
$ deno run -A https://code4fukui.github.io/address-japan/downloadAll.js
```
上記のコマンドを実行すると、日本全国の都道府県、市区町村、町字データがCSVファイルとしてダウンロードされます。

## データ・API
このデータの出典は「アドレス・ベース・レジストリ（市区町村マスターデータ）」(デジタル庁)です。
